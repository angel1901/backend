## Description

This is the backend for the services asociated with the ecommerce project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Add to .env

Values are in the document

```bash
# database
$ DATABASE_URL

# token_auth
$ JWT_SECRET

```

## Required

```
Node v21.6.2
```
